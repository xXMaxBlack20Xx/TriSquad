# .ai Workspace

This folder is used by terminal AI agents.

- `BOARD.md` tracks active work.
- `CONTEXT.md` gives project context.
- `DECISIONS.md` records architectural decisions.
- `tasks/` stores task prompts.
- `outputs/` stores agent results.
- `status/` stores agent status.
- `handoffs/` stores agent-to-agent handoffs.
- `logs/` stores command logs.
- `decisions/` stores expanded decision records when table entries are too small.
- `releases/` stores release notes and trace files.
