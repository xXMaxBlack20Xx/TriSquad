# Release v0.2.0: TriSquad Launcher

## Date

2026-04-24

## Summary

Adds the reusable `trisquad` command and turns the repository documentation into
a user-facing guide for the multi-agent project launcher.

## Included Work

- `scripts/ai-project-init` initializes any git repository with the TriSquad AI
  workspace.
- `scripts/trisquad` moves to the git root, initializes missing AI context, and
  launches Claude Squad.
- `scripts/install-trisquad` installs the reusable template and launcher
  commands into the user's home directory.
- The reusable project template now includes `.ai/README.md`, profile-aware
  board columns, verification placeholders, and `GEMINI.md -> AGENTS.md`.
- `README.md` now explains the purpose, installation, usage, generated files,
  agent roles, delegation flow, release history, and verification commands.

verification. Run `trisquad` manually inside a git repository for the visual
